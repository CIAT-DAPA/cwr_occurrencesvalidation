COORDINATES VALIDATION RASTER

V.1. December 2014
This dataset is a raster without the major lakes, reservoirs in the world, and was created to validate coordinates and define if these coordinates are into the land, coastline or the ocean.
Creator: Chrystian C. Sosa

Sources:
This raster was created using  the nature earth land, rivers, Lake Centerlines, Lakes + Reservoirs and  Coastline polygon features from (http://www.naturalearthdata.com/downloads/110m-physical-vectors/) 1:110m and GSHHG dataset (http://www.ngdc.noaa.gov/mgg/shorelines/gshhs.html).

Fwatures:
Resolution per pixel:~1Km
Coordinates system: WGS 1984

Values:
1: Land
2: coastline
